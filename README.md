<h2>🚀 Sobre o Projeto</h2>
<p>
  Este projeto apresenta uma reanálise detalhada do desempenho financeiro dos filmes do <b>Universo Cinematográfico da Marvel (MCU)</b>. 
  Anos após um estudo inicial, decidi revisitar este conjunto de dados para aplicar novas técnicas de <i>Data Wrangling</i> e criar visualizações mais intuitivas e profissionais.
</p>

<h2>🛠️ Tecnologias Utilizadas</h2>
<ul>
  <li><b>Python:</b> Processamento de dados.</li>
  <li><b>Pandas:</b> Limpeza e manipulação de DataFrames.</li>
  <li><b>Matplotlib:</b> Visualização de dados customizada.</li>
  <li><b>Google Colab:</b> Ambiente de desenvolvimento.</li>
</ul>

<h2>🧹 Destaques do Tratamento de Dados (Data Wrangling)</h2>
<p>Nesta nova versão, priorizei a qualidade e a usabilidade dos dados:</p>
<ul>
  <li><b>Localização:</b> Tradução completa dos títulos dos filmes para o português.</li>
  <li><b>Engenharia de Dados:</b> Conversão de strings monetárias em formatos numéricos para cálculos precisos.</li>
  <li><b>Identidade Visual:</b> Implementação de <i>Color Mapping</i> para diferenciar as fases do MCU no gráfico.</li>
</ul>
